# 알림 카테고리

채널에 전송되는 알림의 카테고리 분류 기준입니다.

## 카테고리 목록

### 🟢 신규 상장 (listing)
- **설명**: 새로운 코인 상장 공지
- **키워드**: `신규 상장`, `신규상장`, `디지털 자산 추가`
- **예시**:
  ```
  🟢 #신규 상장 [업비트]
  테스트코인(TEST) 신규 상장
  https://upbit.com/...
  ```

### 🟢 마켓 추가 (market_add)
- **설명**: 기존 코인의 원화/BTC 마켓 추가, 거래 페어 추가
- **키워드**: `원화 마켓 추가`, `마켓 추가`, `KRW 마켓`, `BTC 마켓`, `거래 페어 추가`
- **예시**:
  ```
  🟢 #마켓 추가 [빗썸]
  패브릭 프로토콜(ROBO) 원화 마켓 추가
  https://feed.bithumb.com/...
  ```

### 🔴 거래유의 (delist_warning)
- **설명**: 거래유의종목 지정 경고
- **키워드**: `거래유의종목 지정`, `거래유의종목지정`, `투자유의`, `유의종목 지정`, `거래유의`
- **예시**:
  ```
  🔴 #거래유의 [업비트]
  비너스(XVS) 거래유의종목 지정
  https://upbit.com/...
  ```

### 🟡 유의종목 해제 (delist_warning_release)
- **설명**: 거래유의종목 지정 해제
- **키워드**: `유의종목 해제`, `거래유의 해제`, `투자유의 해제`
- **예시**:
  ```
  🟡 #유의종목 해제 [빗썸]
  테스트코인(TEST) 유의종목 해제
  https://feed.bithumb.com/...
  ```

### 🔴 상장폐지 (delist)
- **설명**: 거래 지원 종료 (상폐) 확정 공지
- **키워드**: `거래 지원 종료`, `상장폐지`, `마켓 유의종목 해제 및 거래지원종료`
- **예시**:
  ```
  🔴 #상장폐지 [업비트]
  테스트코인(TEST) 거래 지원 종료
  https://upbit.com/...
  ```

### 🟡 입출금 중단 (deposit_suspend)
- **설명**: 네트워크 점검, 업그레이드 등으로 인한 입출금 일시 중단
- **키워드**: `입출금 중단`, `입출금 일시 중단`, `입출금 중지`, `출금 중단`, `입금 중단`
- **예시**:
  ```
  🟡 #입출금 중단 [업비트]
  모나드(MON) 입출금 일시 중단 안내
  https://upbit.com/...
  ```

### 🟢 입출금 재개 (deposit_resume)
- **설명**: 중단되었던 입출금 서비스 재개
- **키워드**: `입출금 재개`, `입출금 정상화`, `입금 재개`, `출금 재개`, `완료`
- **예시**:
  ```
  🟢 #입출금 재개 [빗썸]
  루나2(LUNA2) 출금 일시 중단 안내 (완료)
  https://feed.bithumb.com/...
  ```

### 🟡 네트워크 (network)
- **설명**: 메인넷 스왑, 네트워크 전환, 하드포크 등
- **키워드**: `네트워크 전환`, `네트워크 업그레이드`, `메인넷 스왑`, `메인넷 전환`, `하드포크`
- **예시**:
  ```
  🟡 #네트워크 [업비트]
  테스트코인(TEST) 네트워크 전환 안내
  https://upbit.com/...
  ```

### 🎁 에어드랍 (airdrop)
- **설명**: 에어드랍 지급 공지
- **키워드**: `에어드랍`, `에어드롭`, `airdrop`
- **예시**:
  ```
  🎁 #에어드랍 [빗썸]
  마이쉘(SHELL) 에어드랍 이벤트
  https://feed.bithumb.com/...
  ```

### 🎉 이벤트 (event)
- **설명**: 거래 이벤트, 프로모션, 캠페인
- **키워드**: `이벤트`, `프로모션`, `캠페인`, `경품`, `리워드`
- **예시**:
  ```
  🎉 #이벤트 [업비트]
  Tron Week 총 상금 약 23억 원 이벤트
  https://upbit.com/...
  ```

### 💎 스테이킹 (staking)
- **설명**: 스테이킹, 락업 프로그램 오픈
- **키워드**: `스테이킹`, `락업`, `스테이크`
- **예시**:
  ```
  💎 #스테이킹 [빗썸]
  테스트코인(TEST) 스테이킹 프로그램 오픈
  https://feed.bithumb.com/...
  ```

### 📢 기타 (etc)
- **설명**: 위 카테고리에 해당하지 않는 일반 공지
- **키워드**: 없음 (매칭되지 않은 공지)
- **예시**:
  ```
  📢 #기타 [업비트]
  유통량 계획표 신규 안내 : 인터넷컴퓨터(ICP)
  https://upbit.com/...
  ```
